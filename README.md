# Specsview

**Specsview** is a fast, lightweight, and open-source system information tool designed to give you clear, detailed hardware specifications at a glance.
Built as a modern alternative to utilities like CPU-Z, Specsview focuses on simplicity, speed, and accuracy—without the clutter.

---

## 🚀 Features

* **CPU information** – model, cores/threads, clock speeds
* **GPU details** – vendor, model, driver info
* **Memory overview** – RAM capacity, speed, usage
* **Motherboard & BIOS data** – manufacturer, version, chipset
* **System summary** – OS, build number, kernel, temperatures
* **Modern look** – Pleasing to the eye and fresh.
* **Lightweight & fast** – minimal resource usage
* **Open source** – transparent and community-driven
